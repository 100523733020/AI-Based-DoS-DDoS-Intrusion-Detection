# AI-Based Multi-Class DoS & DDoS Intrusion Detection System

## Overview

This project presents an AI-powered Intrusion Detection System (IDS) capable of detecting and classifying 12 different DoS and DDoS attack types using a hybrid CNN-BiLSTM deep learning architecture with a self-attention mechanism.

The model combines deep learning, class balancing, and explainable AI to achieve high detection performance on network traffic data.

---

## Features

- Multi-class intrusion detection
- CNN-BiLSTM hybrid architecture
- Self-Attention mechanism
- SMOTE for class balancing
- SHAP explainability
- Random Forest & XGBoost baseline comparison
- 12 attack category classification
